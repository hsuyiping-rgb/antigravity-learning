# 開工流程

當使用者說「開工」或「我來了」時，請執行以下步驟：

1. **確認 Git 狀態**：
   - 執行 `git status` 與 `git log -n 5`，確認當前本地工作目錄與遠端分支同步狀況。
2. **讀取 Obsidian 每日筆記**：
   - 獲取今天的日期格式（例如：`2026-07-04`）。
   - 讀取 `G:\我的雲端硬碟\secondbrain\每日筆記\<YYYY-MM-DD>.md`。
   - 如果今天的文件不存在，則嘗試讀取昨天的每日筆記或最近一天的每日筆記。
   - 解析其中的「上次做到哪」與「下一步計畫」。
3. **今日行動建議**：
   - 將讀取到的計畫與 Git 狀態結合，為使用者提供今日的第一步行動建議。
