# 🚀 Antigravity 2 專案駕駛艙 (Project Cockpit)

本專案已連結全平台自動化服務。您可以使用以下關鍵字與虛擬助教互動：

## 🟢 核心指令集 (SOP)
*   **「開工」/「我來了」**：助教將為您檢查當前 Git 狀態並自動讀取 Obsidian `每日筆記` 以提供下一步建議。
*   **「收工」/「下班了」**：助教將自動進行專案安全性掃描、Git 存檔與推送，並將進度記錄回 Obsidian 的 `每日筆記`。
*   **「初始化專案」**：一鍵建立專案結構、Git 倉庫、GitHub 遠端儲存庫與 Obsidian 專案目錄。

## 🌐 連結的服務狀態
*   **Google NotebookLM**：已啟動 (MCP: `notebooklm`)
*   **Obsidian 第二大腦**：已啟動 (MCP: `obsidian` -> `G:\我的雲端硬碟\secondbrain`)
*   **Firebase 雲端部署**：已啟動 (使用 `firebase-tools`)
*   **GitHub 版本控制**：已啟動 (使用 `gh` CLI)
